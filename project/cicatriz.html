Cicatriz

Cicatriz is a project carried out in a school setting. The goal was to create a website featuring data visualization using graphical representations. The tools used to complete this project were open to choice, which is why multiple languages were employed. The creation of this website was divided into four stages: identifying the topic, collecting the data, designing the site layout, and coding.

The website was built without a CMS, in collaboration with another developer and a designer. No software was used, but JavaScript libraries such as Chart.js were utilized. Languages like HTML, CSS, and JavaScript were also employed.
